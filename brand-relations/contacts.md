# Quan hệ Brand / Shop

## Danh sách Brand đang hợp tác
| Brand | Ngành | GMV đã tạo | Hoa hồng hiện tại | Người liên hệ | Ghi chú |
|-------|-------|-----------|------------------|--------------|---------|
| | | | % | | |

## Brands cần liên hệ đề xuất Target Collaboration
> Khi GMV cho 1 seller > $5,000 → liên hệ đề xuất hợp tác

| Brand | GMV hiện tại | Đề xuất | Trạng thái |
|-------|-------------|---------|-----------|
| | | Hoa hồng cao hơn / Mã giảm giá / Free sample | |

## Template email/message đề xuất hợp tác
```
Xin chào [Tên Brand],

Mình là [Tên] – TikTok creator trong lĩnh vực [Niche].
Trong [X tháng] vừa qua, mình đã tạo ra [GMV] cho shop của bạn
thông qua [số] video với tổng [views] lượt xem.

Mình muốn đề xuất hợp tác Target Collaboration để cùng nhau
tăng trưởng doanh thu. Mình mong muốn:
- Hoa hồng: [X]%
- Mã giảm giá độc quyền cho followers
- Free sample để tạo thêm content

Mình có thể cung cấp data và case study chi tiết nếu bạn quan tâm.

Trân trọng,
[Tên]
```
