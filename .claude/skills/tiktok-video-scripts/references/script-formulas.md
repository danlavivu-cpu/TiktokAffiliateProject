# 5 Công thức kịch bản TikTok Affiliate

## Format 1: UNBOXING + FIRST IMPRESSION

### Cấu trúc
```
HOOK (0-3s): Cầm sản phẩm, tạo tò mò
  → Curiosity: "Vừa đặt cái này vì thấy TikTok ai cũng khen..."
  → Shock: "Cái này nhìn như X nhưng mà... không phải đâu!"

PROBLEM (3-8s): Nỗi đau liên quan
  → "Mình hay [hành vi] mà cứ lo [hậu quả]"

SOLUTION (8-15s): Giới thiệu sản phẩm
  → Đặt sản phẩm lên bàn, show nhãn, đọc thành phần/tính năng

DEMO (15-40s): Mở hộp chi tiết
  → Vặn nắp, ngửi, sờ, nếm (nếu food), show bao bì, đọc specs
  → Cảm nhận ban đầu (chân thật)

CTA (40-60s): Follow để xem review
  → "Review chi tiết sau vài ngày dùng, follow nhé!"
```

### Ghi chú production
- Setting: Cafe, bàn gỗ, ánh sáng tự nhiên
- Props: Sản phẩm chưa mở, background sạch
- Mục tiêu chính: Tăng follow + save

## Format 2: HONEST REVIEW (Sau khi dùng)

### Cấu trúc
```
HOOK (0-3s): Cầm sản phẩm đã dùng
  → "Dùng [sản phẩm] được X thời gian rồi, phải nói thật là..."
  → Để câu lơ lửng → người xem muốn biết "thật là" gì

PROBLEM (3-8s): Vấn đề trước khi dùng
  → "Trước mình hay [vấn đề], mà [hậu quả cảm xúc]"

SOLUTION (8-15s): Bắt đầu dùng sản phẩm
  → Kể ngắn gọn lý do bắt đầu dùng

DEMO (15-40s): Review 3+1
  → 3 điểm thích (cụ thể, có dẫn chứng)
  → 1 điểm nhỏ chưa thích (tạo authentic, không deal-breaker)
  → Ai phù hợp, ai không

CTA (40-60s): Link mua hàng
  → "Link ở giỏ hàng màu vàng nhé!"
```

### Ghi chú production
- Setting: Nhà thật, bếp, phòng — authentic
- Props: Sản phẩm đã dùng (có dấu hiệu sử dụng thật)
- Mục tiêu chính: Build trust, convert mua hàng

## Format 3: BEFORE/AFTER hoặc SO SÁNH

### Cấu trúc
```
HOOK (0-3s): Visual shock
  → Đặt 2 sản phẩm/kết quả cạnh nhau
  → "Không ai tin cho đến khi thấy kết quả này!"

PROBLEM (3-8s): Vấn đề với lựa chọn cũ
  → "X rẻ nhưng [nhược điểm dài hạn]"

SOLUTION (8-15s): Giới thiệu lựa chọn mới
  → "Y nghe có vẻ đắt hơn nhưng [lợi ích dài hạn]"

DEMO (15-40s): So sánh trực quan
  → Bảng so sánh (viết tay hoặc text overlay)
  → Nếm/thử/demo cả 2 → kết luận
  → Tính toán chi phí dài hạn

CTA (40-60s): So sánh giá trị
  → "Chênh lệch X lần tiền mà [lợi ích] — link bên dưới!"
```

### Ghi chú production
- Props: Cả 2 sản phẩm để so sánh, bảng so sánh nhỏ
- Text overlay: "TRƯỚC ❌ vs SAU ✅" hoặc "X VND vs Y VND"
- Mục tiêu chính: FOMO, viral, share

## Format 4: TUTORIAL / HƯỚNG DẪN SỬ DỤNG

### Cấu trúc
```
HOOK (0-3s): Education shock
  → "90% mọi người dùng [sản phẩm] sai cách này!"
  → "X cách dùng [sản phẩm] ít ai biết"

PROBLEM (3-8s): Cách dùng sai phổ biến
  → Diễn lại cách sai → "Vừa [hậu quả] vừa [lãng phí]"

SOLUTION (8-15s): Giới thiệu cách đúng
  → "Dùng [sản phẩm] kết hợp đúng kỹ thuật mới có tác dụng"

DEMO (15-40s): Step-by-step
  → Bước 1: [Cụ thể]
  → Bước 2: [Cụ thể]
  → Bước 3: [Cụ thể]
  → Tips bonus / Mẹo độc quyền

CTA (40-60s): Save + mua
  → "Save lại để dùng khi cần! Link ở giỏ hàng."
```

### Ghi chú production
- Props: Đủ để demo từng bước
- Text overlay: Đánh số bước, highlight tips
- Mục tiêu chính: Save + share (evergreen content)

## Format 5: STORYLINE / ĐỜI THƯỜNG

### Cấu trúc
```
HOOK (0-3s): Emotional hook
  → "Mình suýt bỏ cuộc cho đến khi..."
  → "Mình suýt [hành động tiêu cực] vì [nỗi đau]"

PROBLEM (3-15s): Câu chuyện cá nhân
  → Kể chuyện thật, chi tiết cụ thể
  → Tạo đồng cảm "đúng là mình luôn!"
  → Đỉnh điểm cảm xúc

SOLUTION (15-30s): Sản phẩm xuất hiện tự nhiên
  → "Rồi bạn mình giới thiệu cái này..."
  → KHÔNG giới thiệu như quảng cáo

DEMO (30-45s): Kết quả trong câu chuyện
  → "Từ hôm đó, [thay đổi tích cực]"
  → Visual proof nếu có

CTA (45-60s): Empathy CTA
  → "Nếu bạn cũng từng như mình — thử đi sẽ không hối hận đâu!"
```

### Ghi chú production
- Setting: Đời thực hoàn toàn (cafe, nhà, đường phố)
- Không nhìn script, kể chuyện tự nhiên
- Cảm xúc cần truyền đạt: chân thành, ấm áp
- Mục tiêu chính: Viral cao nhất (share + comment)
