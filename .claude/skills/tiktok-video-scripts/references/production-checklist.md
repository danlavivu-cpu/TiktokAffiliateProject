# Production Checklist — TikTok Affiliate Video

## Trước khi quay

- [ ] Script đã duyệt (HOOK→PROBLEM→SOLUTION→DEMO→CTA)
- [ ] Hook type xác định (Curiosity/Personal/Social proof/Education/Emotional)
- [ ] Background sạch, phù hợp niche
- [ ] Outfit phù hợp (không logo brand khác)
- [ ] Ánh sáng tốt (tự nhiên hoặc ring light)
- [ ] Sản phẩm + props sẵn sàng
- [ ] Link affiliate đã active trên TikTok Shop
- [ ] Điện thoại sạc đầy, bộ nhớ đủ

## Khi edit

- [ ] Hook đủ mạnh trong 3s đầu (50% tác động nằm ở đây)
- [ ] Text overlay rõ ràng, đọc được trên mobile
- [ ] Trending sound/nhạc nền phù hợp mood
- [ ] Độ dài 30-60s (sweet spot cho TikTok)
- [ ] CTA rõ ràng, có text chỉ đường mua
- [ ] Không có watermark app khác
- [ ] Tỷ lệ 9:16 (vertical)

## Trước khi đăng

- [ ] Caption + hashtags (5-10 tags)
  - 2-3 hashtags sản phẩm/brand
  - 2-3 hashtags niche
  - 2-3 hashtags trending
  - 1-2 hashtags CTA
- [ ] Link sản phẩm gắn vào giỏ hàng
- [ ] Đúng giờ peak: 11h-13h hoặc 19h-22h
- [ ] Ghi vào `data/product-performance.csv`

## Sau khi đăng (1h đầu quan trọng)

- [ ] Trả lời comment trong 1 giờ đầu
- [ ] Pin comment có giá trị (hỏi sản phẩm = quan tâm mua)
- [ ] Check analytics sau 2-4 giờ
- [ ] Nếu views > 10K + có đơn → cân nhắc Spark Ads
- [ ] Nếu 3 video liên tiếp < 500 views → dừng sản phẩm
